Cinema Simulator

Java · JavaFX · MVC · Desktop app

What it is

A desktop cinema booking system built with JavaFX. Users can browse sessions, purchase tickets, food, and drinks using a simulated account balance, with full validation for edge cases like insufficient or excessive amounts.

What I built

Designed the full application using MVC architecture — keeping UI, business logic, and data clearly separated. Implemented the account balance system with purchase validation and automatic balance updates across all transaction types.

What I learned

MVC forces you to think about what belongs where before you write a single line. I found myself refactoring a lot early on when I realised I'd put logic in the wrong layer — which was a frustrating but genuinely useful lesson.
